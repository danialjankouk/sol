Solidity Study Repository
Overview

This repository is a simple personal workspace for learning Solidity and Ethereum smart contracts.

It focuses on understanding core concepts, writing small examples, and gradually getting familiar with common patterns and security basics.

Goals

Learn Solidity fundamentals

Understand how smart contracts work on Ethereum

Practice writing and reading smart contract code

Get basic awareness of common security issues

Notes

Contracts here are for learning only

Not audited

Not meant for production use
